#Instagram
==========

This is my second project with Ruby on Rails.

Technologies used
-----------------
For the completion of this project I used the following technologies:

* Ruby on Rails
* Rspec
* Capybara
* Devise
* Paperclip
* PostgreSQL
* HTML5
* CSS3
